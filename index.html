Metro+

This project aims to allow the control and visualization of the location for the underground train lanes through an application that was developed in JavaFX. I have also used Arduino, in which each train would have a sensor that when passing through a certain point it sends the information to the program in JSON which then updates the location on the dashboard. The administrator will have access to all resources, these being a dashboard with all the information of the trains and respective lanes, status and location of the trains and statistics of users and stations. Users will have access to an application in which they can visualize schedules, arrival times at stations and lane operating status. This project was born from the need to respond to the dissatisfaction of people who use the subway trains service on a daily basis, this being the lack of information outside the train stations. Currently, there is no way to know when the trains are delayed or if the lanes are eventually out of service. The purpose of this project is to provide the population with a service that allows access in real time to schedules, arrivals at stations and status of lanes.
